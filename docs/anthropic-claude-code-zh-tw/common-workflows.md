## 恢復先前的對話

假設您一直在使用 Claude Code 處理任務，需要在稍後的會話中從中斷的地方繼續。

Claude Code 提供兩個恢復先前對話的選項：

- `--continue` 自動繼續最近的對話
- `--resume` 顯示對話選擇器

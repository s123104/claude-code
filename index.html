❓ 常見問題與疑難排解

安裝與啟動問題

  • Node.js 版本需 18+，建議用 LTS
  • WSL 請用 Linux npm/node，勿用 Windows 版
  • pip/pipx/uv 安裝 claude-monitor，PATH 問題請加 ~/.local/bin
  • 首次啟動需完成驗證，支援多平台

權限與安全問題

  • 所有敏感操作需明確批准，建議啟用 OpenTelemetry 監控
  • 預設僅開啟 View 權限，機密資訊用環境變數
  • 避免於生產環境使用 --dangerously-skip-permissions

常見錯誤與排查

  • Rate limit exceeded：升級方案或減少請求
  • 權限錯誤：allowedTools 設定錯誤
  • MCP 問題：MCP 未啟動或異常，請重啟
  • Linux 權限：npm 前綴設家目錄，避免 sudo

效能與最佳實踐

  • 大型專案建議分目錄分析，適當設定 --output-format、--timeout
  • 自動化批次審查、管道整合、結構化輸出
  • 定期 /compact，忽略大型目錄可降資源消耗
詳細疑難排解請參閱 claude-code-guide-zh-tw.md、claude-code-usage-monitor-zh-tw.md、superclaude-zh-tw.md。

快速安裝指南

只需幾個簡單步驟，即可在您的開發環境中啟用 Claude Code 智慧助手。

系統需求

作業系統

  • macOS 10.15+
  • Ubuntu 20.04+ / Debian 10+
  • WSL (Windows Subsystem for Linux)

硬體與軟體

  • 至少 4GB RAM
  • Node.js 18+
  • git 2.23+ (選用)
  • 網路連線

安裝步驟

1

安裝 Claude Code

npm install -g @anthropic-ai/claude-code
2

啟動並驗證

cd your-project-directory
claude

首次啟動會引導您完成驗證設定

3

開始使用

> what does this project do?
> add a hello world function
> commit my changes
